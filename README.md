# IMC-DADOS-DO-Nathan
# Calculadora de IMC

Uma aplicação simples e funcional para calcular o **Índice de Massa Corporal (IMC)**.

## Sobre o Projeto

Este projeto consiste em uma calculadora de IMC desenvolvida com tecnologias web básicas e um servidor Python leve.  
Não utiliza frameworks como Flask, Django ou qualquer outra biblioteca externa.

## Arquivos do Projeto

| Arquivo          | Descrição |
|------------------|---------|
| `app.py`         | Servidor web simples que permite acessar a aplicação no navegador |
| `index.html`     | Página principal com formulário e interface do usuário |
| `style.css`      | Arquivo de estilos (CSS) separado para melhor organização |

## Como Executar

1. **Coloque os três arquivos** (`app.py`, `index.html` e `style.css`) na **mesma pasta**.

2. Abra o terminal ou prompt de comando na pasta do projeto.

3. Execute o seguinte comando:
   ```bash
   python app.py
